Storing Simple Numbers

To begin with we need to look at how to store basic data. We can name our data as anything we want, so long as it doesn’t start with a number. In this case we are using a and storing a 1 in that point.

The a is not case sensitive and you can enter it as an upper case or lower case value. It can be anything that you would like to use. Use descriptive names. Quick tip, if you use consistent names your life can be easier.

There are two ways that we can enter the data. The first way is using <- to enter the number. Using the = sign also works, but due to issues in the base langauge (S plus) for R, it can throw errors. Best coding practices are to use = within functions if you create them and the <- to assign data.

Last is the number you enter. If you are assigning a single number you can just put that number. If you are putting multiple things in you would use c (for Combine) and put commas between the values.

If you want to store text, you need to put the text between either single quotes or double quotes. Both examples are listed here.

To recall the data that is stored, you just need to enter the name that you stored it under, in this case a, b, c, or d. Go ahead and try it.
